# 一般社団法人あんだんて — 公式サイト（リビルド版）

福島県白河市・西郷村エリアで地域住民の憩いの場づくりを行う **一般社団法人あんだんて** の公式サイトの完全リビルド版です。

## このサイトについて

オリジナルの WordPress サイト ([s-andante.org](https://s-andante.org)) を元に、以下を行った静的サイトです:

- **393件の投稿** — 全文・画像・カテゴリ込み
- **9つの固定ページ** — 4つの主要事業ページ含む
- **506MB のメディアアセット** (380枚 + 178件の本文埋込) — 画像・PDF・動画
- **モダンなレスポンシブデザイン** — Helvetica + 明朝の和洋折衷

## 4つの主要事業

| 事業 | 概要 |
|---|---|
| **らふみーる こども＆みんなの食堂** | 月3-4回開催。地域の子どもたち・保護者・住民が集う食事の場 |
| **きららかサロン** | 地域住民の憩いの場、お茶を飲みながらの交流空間 |
| **ユースプレイス県南** | 若者の居場所、第三の居場所づくり |
| **子育て相談室** | 子育てに関する不安・悩みへの相談対応 |

## サイト構造

```
/                       — ホーム（ヒーロー＋4事業＋最新ニュース12件）
/news/                  — ニュース一覧（30件/ページ × 14ページ）
/category/{slug}/       — カテゴリ別アーカイブ（7カテゴリ）
/{year}/{month}/{day}/{slug}/ — 個別投稿ページ
/{page-slug}/           — 固定ページ
/media/                 — 画像・PDF・動画
/assets/                — CSS
```

## 再生成

オリジナルから再取得・再生成する場合:

```bash
node generate.js
```

`posts_full.json` `pages_full.json` `categories.json` `media_mapping.json` を入力として全HTMLを再生成します。
